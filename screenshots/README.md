# Screenshots

Referenced from the top-level [`README.md`](../README.md). Captured on an Amazon Echo Show 8
(1280×800) running the receiver. All content is **synthetic** — the cover is a generated gradient
and the track, artist, listener and lyrics are placeholders (no copyrighted material).

| File | Shows |
| --- | --- |
| `idle.png` | Ambient idle clock face (nothing playing) |
| `now-playing.png` | Now-playing screen — album-art wash, accent color, transport |
| `lyrics.png` | Synced lyrics, active line highlighted |
| `settings.png` | Settings (device name, bitrate, fullscreen, 24h time) |
| `session.png` | Session & receiver health sheet (connected listener, discovery) |

Capture tip: `adb exec-out screencap -p > screenshots/now-playing.png`, then crop to the screen bounds.
